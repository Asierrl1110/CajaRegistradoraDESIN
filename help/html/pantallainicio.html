Funcionalidades de Inicio

La interfaz de inicio permite a los usuarios autenticarse y acceder a la aplicación mediante un formulario sencillo. Las principales funcionalidades disponibles en esta sección son:

1. Iniciar Sesión

Los usuarios pueden iniciar sesión proporcionando su nombre de usuario y contraseña. A continuación, se describe el proceso:

  1. Ingrese su nombre de usuario en el campo correspondiente.
  2. Ingrese su contraseña en el segundo campo.
  3. Haga clic en el botón "Iniciar Sesión".

Si las credenciales son correctas, el sistema mostrará un mensaje de éxito y abrirá la pantalla principal de la aplicación. Si no lo son, se generarán mensajes de error indicando:

  • Usuario no existente: Si el nombre de usuario no está registrado.
  • Contraseña incorrecta: Si la contraseña no coincide con la del usuario registrado.
  • Campos vacíos: Si el usuario o la contraseña no han sido completados.

Si el registro es exitoso, el sistema mostrará un mensaje de confirmación y almacenará la información del nuevo usuario.

3. Componentes de la Pantalla de Inicio

La pantalla de inicio incluye los siguientes elementos:

  • Campo Usuario: Un espacio donde los usuarios pueden ingresar su nombre de usuario.
  • Campo Contraseña: Un espacio para ingresar la contraseña.
  • Botón Iniciar Sesión: Inicia el proceso de validación de las credenciales del usuario.

4. Personalización

La interfaz utiliza un diseño básico pero personalizado:

  • El botón "Iniciar Sesión" está configurado con un borde blanco, fondo gris, y texto blanco.
  • El sistema muestra mensajes de error y éxito mediante cuadros de diálogo emergentes (JOptionPane).

Nota

Los usuarios deben asegurarse de que sus credenciales sean correctas. Si no recuerdan su contraseña, deben comunicarse con el administrador del sistema.
