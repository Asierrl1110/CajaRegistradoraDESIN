Funciones de Administrador

Los usuarios con permisos de administrador tienen acceso a herramientas y opciones avanzadas que les permiten gestionar la aplicación de manera más completa. Estas son las funcionalidades disponibles:

Activar Permisos de Administrador

Para activar las herramientas administrativas:

  1. Acceda al menú "Administrador" en la barra de menú superior.
  2. Seleccione la opción "Activar permisos administrador".
  3. Al activarla, se mostrará una barra de herramientas con opciones adicionales.

Funciones Disponibles

1. Gestión de Usuarios

  • Registrar un usuario: Haga clic en el botón con el icono de usuario en la barra de herramientas. Ingrese un nombre de usuario y una contraseña para completar el registro.

2. Gestión de Productos

  • Crear un producto: Haga clic en el botón con el icono de producto en la barra de herramientas. Esto abrirá una ventana para ingresar los detalles del nuevo producto.

Nota

Estas funciones están disponibles únicamente para usuarios que hayan iniciado sesión como administradores.
