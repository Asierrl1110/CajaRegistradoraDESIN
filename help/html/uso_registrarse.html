Registrarse

El botón de "Registrarse" permite a los nuevos usuarios crear una cuenta en la aplicación. A continuación, se explica el proceso:

Pasos para Registrarse

  1. Haga clic en el botón "Registrarse" en la pantalla principal.
  2. Ingrese el nombre de usuario que desea usar.
  3. Escriba una contraseña segura en el campo correspondiente.
  4. Haga clic en el botón "Crear Cuenta" para finalizar el registro.

Errores Comunes

  • Si el nombre de usuario ya existe, se mostrará un mensaje indicando que debe elegir otro nombre.
  • Si las contraseñas no coinciden, se mostrará un mensaje solicitando que las revise y vuelva a intentarlo.
  • Si deja algún campo vacío, la aplicación le pedirá que complete todos los datos requeridos.

Nota: Una vez registrado, podrá iniciar sesión con el nombre de usuario y contraseña que eligió.
